# Vitraž Page Editor

Vitraž (Serbian for stained glass) is an experimental modular editor inspired by the beauty of interconnected glass panels, where pages are built from blocks that fit together — currently a simple Notion-like starting point while we explore and refine the core architecture.

![Stained glass rose window](https://hips.hearstapps.com/hmg-prod/images/stained-glass-rose-window-in-notre-dame-cathedral-paris-france-1494433394.jpg?crop=1.00xw:0.662xh;0,0.189xh&resize=1400:*)

## Philosophy

A quick preview of Vitraž as the modular pieces start coming together. While the UI currently resembles Notion, the goal is to enable dynamic, interconnected building blocks that share state and behave as a cohesive system.
